#🎓 Ders Kayıt Sistemi#

Bu proje, öğrencilerin derslere kayıt olabildiği, kayıtlarını görüntüleyebildiği ve ders yönetimi işlemlerini gerçekleştirebildiği bir Ders Kayıt Sistemi uygulamasıdır.

Proje, nesne yönelimli programlama (OOP) prensiplerini uygulamak ve temel sistem tasarımı mantığını kavramak amacıyla geliştirilmiştir.

#🚀 Proje Özellikleri#

📚 Ders listeleme

📝 Derse kayıt olma

❌ Dersten kayıt silme

👤 Öğrenci bilgilerini görüntüleme

📋 Kayıtlı dersleri listeleme

🔢 (Varsa) Kontenjan kontrolü

🛠️ Kullanılan Teknolojiler

Java

Nesne Yönelimli Programlama (OOP)

(Varsa ekle: Java Swing / Console Application / Spring Boot)

#📂 Proje Yapısı#

src/ → Kaynak kod dosyaları

model/ → Öğrenci ve Ders sınıfları

service/ → Kayıt işlemleri ve iş mantığı

main → Uygulama başlangıç noktası
